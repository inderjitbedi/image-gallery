Image Gallery Assignment

Make an Angular application having following actions:

  1. Create 6 boxes.
  2. When clicked on a box, it should select and show that image into it.
  3. Similarly each box should show an image which is selected by clicking on that particular box.
  4. Also provide an option to delete the image of a particular box.
  5. Fill box in series.i.e if i click on box 6 and box 3,4,5 is empty then image will show in box 3 and so on.
  6. If all 6 images filled and i removed 2nd image then all images shifts and 6th box become empty.
